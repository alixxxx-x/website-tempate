# interShip! – Internship Management Platform

## Unit
L3TI Workshop  
Academic Year: 2025–2026  

## Project Title
interShip! – Internship Management Platform

---

## Project Description

This is a full-stack web application designed to streamline internship management and facilitate connections between students and companies.  
The platform provides dedicated spaces for students, companies, and university administrators, handling internship applications, validation, and automatic document generation.  

The system allows role-based access, centralized tracking of applications, and skill-based matching between students and companies.  
Administrators can validate internships and generate PDF agreements automatically, while students can apply to offers and companies can manage applications efficiently.  

This project is the **final graduation project** for the Computer Science program, demonstrating full-stack development and deployment skills.

---

## Actors and Roles

### Student
- Register and securely log in  
- Create a digital CV with personal details, technical skills, and portfolio links  
- Search and apply for internship offers with filters (location, skills, type)  
- Track application status  

### Company (Recruiter)
- Manage company profile (logo, description, location)  
- Create, modify, and delete internship offers  
- Track applications and accept or reject candidates  

### Administrator
- Validate internships approved by companies  
- Automatically generate internship agreements and certificates (PDF)  
- Monitor global placement statistics  

---

## Functional Requirements

- Single application and login system for all actors  
- Role-based access control (Student, Company, Administrator)  
- Predefined admin account  
- Students can apply to multiple offers  
- Companies can manage multiple internship offers  
- Automatic document generation after admin validation  
- Skill-based search and filtering for better matching  

---

## Technologies Used

- **Frontend:** ReactJS  
- **Backend:** Django  
- **Database:** PostgreSQL  
- **Authentication:** JWT-based login  
- **Other:** PDF generation for internship agreements
